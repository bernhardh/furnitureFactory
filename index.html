costco

costco is a small UI for bulk editing CouchDB documents. It takes a map function and executes it on all the docs in the database. For convenience it's a couchapp, so you can push it to any db like so: 

git clone http://github.com/harthur/costco.git
cd costco
couchapp push . http://hostname:5984/mydatabase
and visit it at http://hostname:5984/mydatabase/_design/costco/index.html

Examples
change the error field of all docs 
function(doc) {
  doc.error *= 100;
  return doc;
}
delete all docs that have year less than 2010 
function(doc) {
  if(doc.year < 2010)
    return null;
  return doc;
}
rename message field in all docs 
function(doc) {
  doc.text = doc.message;
  delete doc.message;
  return doc;
}
